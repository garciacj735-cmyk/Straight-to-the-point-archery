# Straight-to-the-point-archery

A simple and user-friendly website for archery enthusiasts.  
This website provides information, tips, and resources for both beginners and experienced archers.

## Features

- **Home Page**: Introduction to archery and its benefits
- **Tutorials**: Step-by-step guides for beginners
- **Equipment Guide**: Overview of bows, arrows, and other gear
- **Gallery**: Images of archery practice and competitions
- **Responsive Design**: Works on desktop, tablet, and mobile

## Notes

- All main website files (`index.html`, `style.css`, scripts, and images) should be in the **root directory** for GitHub Pages to display the site correctly.
- GitHub Pages supports only HTML, CSS, and JavaScript. PHP or server-side scripts will not work.
